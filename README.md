## 3. 기본 그래프 그리기

| 기능           | 코드 예제                                             | 설명                       |
|----------------|-------------------------------------------------------|----------------------------|
| 선 그래프      | ```python<br>import matplotlib.pyplot as plt<br>plt.plot([1,2,3], [4,5,6])<br>plt.show()``` | x,y 데이터로 선 그래프 생성 |
| 산점도        | ```python<br>plt.scatter([1,2,3], [4,5,6])<br>plt.show()``` | 점으로 산점도 표현          |
| 막대그래프     | ```python<br>plt.bar(['A','B','C'], [5,7,3])<br>plt.show()``` | 카테고리별 막대그래프 생성  |
| 히스토그램    | ```python<br>plt.hist([1,1,2,3,3,3,4,5])<br>plt.show()``` | 데이터 분포 히스토그램      |

---

## 4. 그래프 꾸미기

| 기능           | 코드 예제                                                   | 설명                               |
|----------------|-------------------------------------------------------------|----------------------------------|
| 제목 추가      | ```python<br>plt.title('그래프 제목')```                     | 그래프 상단 제목 설정             |
| 축 이름        | ```python<br>plt.xlabel('X축')<br>plt.ylabel('Y축')```        | x축, y축 이름 설정                |
| 범례           | ```python<br>plt.plot(x, y, label='데이터1')<br>plt.legend()``` | 범례 표시                         |
| 그리드         | ```python<br>plt.grid(True)```                               | 그래프 배경 그리드 표시           |
| 축 범위 지정   | ```python<br>plt.xlim(0,10)<br>plt.ylim(0,100)```              | 축의 표시 범위 설정               |

---

## 5. 여러 그래프 그리기

| 기능           | 코드 예제                                                     | 설명                           |
|----------------|---------------------------------------------------------------|------------------------------|
| 여러 선 그리기 | ```python<br>plt.plot(x1, y1)<br>plt.plot(x2, y2)<br>plt.show()``` | 한 그래프에 여러 선 그리기    |
| 서브플롯       | ```python<br>fig, axs = plt.subplots(2,1)<br>axs[0].plot(x,y)<br>axs[1].bar(a,b)<br>plt.show()``` | 여러 그래프를 한 화면에 배치  |

---

## 6. 주요 함수 및 옵션 정리

| 함수명               | 설명                                  | 주요 옵션 예시                             |
|----------------------|-------------------------------------|-------------------------------------------|
| `plt.plot()`          | 선 그래프 생성                      | `color='red'`, `linestyle='--'`, `marker='o'` |
| `plt.scatter()`       | 산점도 생성                        | `c='blue'`, `s=50`                        |
| `plt.bar()`           | 막대그래프 생성                    | `width=0.5`, `color='green'`              |
| `plt.hist()`          | 히스토그램 생성                    | `bins=10`, `alpha=0.7`                    |
| `plt.title()`         | 그래프 제목 설정                   | -                                         |
| `plt.xlabel()`, `ylabel()` | 축 이름 설정                    | -                                         |
| `plt.legend()`        | 범례 표시                         | -                                         |
| `plt.grid()`          | 그리드 표시                       | -                                         |
| `plt.savefig()`       | 그래프 이미지 저장                | `dpi=300`, `format='png'`                  |

---

## 7. 저장 및 출력

| 기능            | 코드 예제                           | 설명                  |
|-----------------|-----------------------------------|-----------------------|
| 그래프 저장     | ```python<br>plt.savefig('graph.png', dpi=300)``` | 고해상도 이미지 저장  |
| 화면 출력      | ```python<br>plt.show()```          | 그래프 화면에 출력     |
